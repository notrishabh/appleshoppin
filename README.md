# Apple Watch Studio Clone 🍓

This project is a clone of the [Apple Watch Studio page](https://www.apple.com/shop/studio/apple-watch), enabling users to customize their Apple Watch by selecting various cases, bands, and sizes and sharing it.

## ✨ Features

- **Diverse Watch Collections**: Choose from various collections, including Apple Watch Series 10, Apple Watch Hermès Series 10, and Apple Watch SE.
- **Case Size Options**: Select the ideal case size to fit your wrist perfectly.
- **Customizable Cases**: Personalize your watch with multiple cases options, including Aluminum and Titanium.
- **Customizable Bands**: Personalize your watch with a wide selection of bands.
- **Side View Previews**: Explore detailed side profiles of your customized watch (available for certain models).
- **Seamless Customization Preview**: See changes in real-time as you make selections.
- **Real-time price calculation**: Check the final price of your customized watch.
- **Sharing**: Share your customized watch with your friends and socials or download an image.
- **Responsive Layout**: Ensures a seamless experience across all devices and screen sizes.
- **Smooth Animations & Transitions**: Enjoy fluid and captivating interactions throughout the experience.
- **Optimized for Speed**: A fast and efficient customization process for a smooth user experience.
- **Cross-Browser Support**: Works consistently across all major browsers.
- **Fully Accessible**: Built with full ARIA support to ensure accessibility for all users.
- **Extensive Documentation**: Comprehensive documentation to guide you through the setup and usage process.

## ⚡️ Requirements

- Node >= **16.14.0**
- NPM >= **10.8.3**
- Git >= **2.19.0**

## 🚀 Getting Started

To get started, clone the repository and install the necessary dependencies:

    git clone https://github.com/notrishabh/appleshoppin.git
    cd appleshoppin
    npm install

## 💡Usage

To run the project locally, use the following command:

    npm run dev

This will start the development server at http://localhost:3000.
